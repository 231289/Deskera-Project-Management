Deskera Project Management Change Log
Version 0.5
Release Date: 29 Oct 09
changes
1. Baseline Comparison - A new Component for Comparing saved Baselines with current project plan has been added.
It shows-although not like 'diff'- color coded comparison between each task with respect to Baseline.

At Project Plan -> Baseline -> Compare Baselines

2. Gantt Chart Scaling to Week View- Users can now switch seamlessly between Day and Week View of the Gantt Chart

At Project Plan -> View Selection Combo box -> Week View

3. Improved To-Do interface - Now users can have simplified yet featured To-do lists as they now have priority, Grouping, due dates and description. To-Do assignment to a resource is now optional.

At To-Do -> Add a To-Do -> Double Click on To-Do

4. Documents at Project Level - Share Documents between members and resources at Project level now.
A new sub-tab added on Project level now enables Users to have project specific docs which also exist in My Documents under the Project name Tag

At Dashboard -> Documents Tab

5. Audit Trails - Administrators can now have an eye on activities performed through the Audit Trails. Clicking on the shortcut near the Global Search gives all the activities performed which can be sorted by several fileters provided.

(For Company Admin only)At Dashboard -> Shortcuts before Global Search

6. Added functions on Task Context Menu - Functions for Requesting Task Progress Update and quick notifications to Applied Resources on that Task.

At Project Plan -> Tasks Grid -> Context Menu

7. Several Bug Fixes - Other bug fixes for Personal Messages, Import/Exports, Reports, Plan, Rendering Errors (in IE versions)etc.
Version 0.6
Release Date: 20 Nov 09
changes
1. New Reports - New project reports under Baseline Reports. Provides a Variance between baseline and actual project plan with overall project summary. Some new reports under overview and workload as well.(Reports count is 18 now)

At Project Plan -> Reports

2. Improved Notifications- Users can select to change notification type (e.g. sms, email) and frequency of sending it(daily, weekly, Immediately). (sms facility currently not implemented). Also Calendar now has notification options depending on calendar view and events which are sent to all the involved project members.

At Project -> Team Calendar -> Notifications and At Administration -> Company Administration

3. Task References - Users can add references to the tasks in terms of URL's. A new tab added in Task Modal for doing so. And Clicking on Notes in project plan shows them back.

At Project Plan -> Task Information -> References

4. Import Outlook To-do Tasks - Feature of importing to-do tasks from external csv files included. Options of appending or overwriting to current to-do's has been provided.

At Project -> To-Do -> Import CSV

5. Team Project Relationship - Company Administrators can now create teams of individual resources and assign them onto projects. Clicking on the new link on Dashboard(in quick links) starts a wizard based team creation and assignment. Assigned teams appear under project level administration in resources section.

(For Company Admin only)At Dashboard -> Quick Links

6. Color Coding in Reports - In Project Reports, color coding help in understanding data quickly. Users can select to turn it on/off. Users can also select to change the colors for a particular range. Provided facility for users to even decide the Range for themselves.

At Project Plan -> Reports

7. Others
- iDeskera Changes.
- Bug fix in Embed Project Plan.
- Code Modularization in Project Reports JS's.
- Descriptive Quick Tips/Tool Tips Provided at some places.
